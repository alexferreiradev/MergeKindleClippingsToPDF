Merge Kindle Clips To PDFs

Gosta de criar marcações nos seus PDFs pelo Kindle e não consegue exportar para um PDF e ler no seu PC ? Este projeto foi criado para resolver este problema.

Get started

Os passos para começar a utilizar o Merge, podem ser acessados através deste link. Não é difícil, mas siga os passos para que não ocorra nenhum imprevisto.

Features

O Kindle não exporta PDFs com anotações para outro dispositivo, somente cria um arquivo com as anotações separados do PDF. Devido a esta limitação, o Merge Kindle Clips To PDF realiza:

  • Criação de marcações em PDFs exportados do Kindle.

Restrictions

  1. O Merge Kindle Clips só abre arquivos iguais e com o mesmo nome do arquivo criado dentro do Kindle, o My Clippings.txt.
  2. A versão atual não possui suporte a notas e outros tipos de anotações feitas em um PDF, somente marcações.
